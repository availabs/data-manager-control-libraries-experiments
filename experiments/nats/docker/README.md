# NATS docker

USAGE:

1. Create the TLS certificates

```sh
cd ./ca
createCertificates
```

2. Configure the server: [./config/nats-server.conf](./config/nats-server.conf)

3. Configure the Docker container: [./nats.env](./nats.env)

4. Start the NATS server: `./start`
